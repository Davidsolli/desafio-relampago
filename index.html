SindPro é um sistema acadêmico de gerenciamento de condomínios, criado para facilitar a administração e comunicação entre síndicos e moradores. Desenvolvido com Angular no front-end e Spring Boot no back-end, oferece uma arquitetura moderna, escalável e de fácil manutenção.

Projeto desenvolvido em uma semana como desafio acadêmico, focado em criar um back-end para cadastro e administração de clientes. Construído com Spring Boot, aplicando arquitetura em camadas e persistência de dados. Demonstra boas práticas de desenvolvimento back-end e integração via API REST.

Projeto de landing page desenvolvido com HTML, CSS e JavaScript puro, estruturado de forma modular. Cada componente da página foi separado para facilitar manutenção e reaproveitamento de código. O resultado é uma interface responsiva, organizada e de fácil expansão para novos recursos.

Projeto de resumos acadêmicos desenvolvido em formato Markdown, reunindo conteúdos das aulas de forma organizada. Serve como repositório de estudos, permitindo rápida consulta e revisão dos principais tópicos. A estrutura modular facilita a leitura, manutenção e expansão contínua dos resumos.
